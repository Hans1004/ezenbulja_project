트레이딩뷰 제공 CRYPTOCAP:BTC.D

사용 방법

1. 모델 학습:
  • 기존 데이터를 AI에게 "모델 학습" 버튼을 클릭하여 모델 학습을 시작합니다.
  • 모델 학습이 완료되면, "Training Complete!"라는 문구가 출력됩니다.
2. 코인 정보 입력:
  • 페이지 상단의 입력 필드에 코인 관련 정보를 입력합니다.
  • 입력 필드에는 시가, 종가, 저가, 거래량, 이전 종가, 변동 가격, RSI를 최신정보입력 버튼을 눌러 입력할 수 있습니다.
3. 코인 가격 예측:
  • 예측된 코인 가격이 화면에 표시됩니다.
